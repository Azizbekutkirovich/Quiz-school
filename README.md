# 📚 Quiz School — Digital Testing Platform for Schools

🛠 **Built with PHP Yii2 (Basic Template)**  
📁 **Project structure:**  
- `quiz-school` — Student-side application  
- `quiz-school-teacher` — Teacher-side application

---

## 🎯 Project Purpose

In many schools, exams and tests are still conducted on paper. This leads to significant time and financial costs for both teachers and students. To address this issue, I developed a modern digital testing platform using the **PHP Yii2 framework**.

This project allows:
- 🧑‍🎓 **Students** to take subject-based tests online
- 👩‍🏫 **Teachers** to create and manage tests, and monitor student performance

---

## 🔍 Project Overview

### 👨‍🎓 Student Panel (`quiz-school`)
- Browse and select available subjects
- Take tests and receive immediate results
- User-friendly interface for test-taking

### 👩‍🏫 Teacher Panel (`quiz-school-teacher`)
- Upload and manage test questions
- View student performance and scores
- Simple admin interface for educators

---

## ⚙️ Tech Stack

- PHP (Yii2 Basic Template)
- MySQL (database layer)
- HTML/CSS/Bootstrap (UI)
- MVC Architecture

---

## ✅ Features

- 🔐 User authentication for both students and teachers
- 📝 Multiple-choice test engine
- 📊 Student progress tracking
- 🗂 Subject and question bank management
- 📬 Real-time result display after test completion

---

## 🚀 How to Run

1. Clone the repository
2. Set up virtual host and point to `/quiz-school` and `/quiz-school-teacher`
3. Import the SQL database file into MySQL
4. Update the DB config in `config/db.php`
5. Done! You can now test the app locally.
